# 🍪 팡쫀쿠 (FAANG_JZONKU)

> 두바이 쫀득 쿠키가 아닌 **팡(판)교 쫀득 쿠키!**

개발자들이 **HTML / CSS / JavaScript**를 재미있게 학습할 수 있도록 만든  
**퀴즈 기반 미니 게임 프로젝트**입니다.

퀴즈를 맞추면 **올바른 재료들이 쿠키에 들어가고**  
틀리면 **괴!상!한! 재료들이 들어갑니다.**

정답을 맞춰 **맛있는 팡쫀 쿠를 완성해보세요! 🍪**
